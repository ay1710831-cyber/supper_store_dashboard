🛒 Superstore Dashboard (Power BI)
📌 Overview

The Superstore Dashboard is an interactive data visualization project built using Power BI to analyze sales and business performance. It provides meaningful insights into key metrics such as sales, profit, and category performance, helping users understand overall business trends.

This project demonstrates how raw data can be transformed into a visually appealing and insightful dashboard.

🎯 Objectives
To analyze sales and profit performance
To identify top-performing categories and regions
To understand trends over time
To create an interactive and user-friendly dashboard
🗂️ Dataset

The dataset used in this project contains retail sales information, including:

Order Details
Sales and Profit
Category and Sub-Category
Region and Segment
Date/Time Information
📊 Dashboard Description

The Superstore Dashboard presents data through interactive visualizations such as bar charts, line graphs, pie charts, and KPI cards. It allows users to explore data dynamically using filters and slicers.

Key insights include:

Sales and profit overview
Category and sub-category performance
Region-wise sales comparison
Time-based trend analysis

Microsoft Excel (Data Preparation)
